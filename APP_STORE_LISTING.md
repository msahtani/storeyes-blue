# Storeyes - App Store Listing Information

## App Description

Storeyes is a comprehensive B2B retail monitoring and management application designed for store owners and managers to monitor operations, track performance, and manage expenses in real time.

### Key Features:

**• Alert Management**
View and manage real-time alerts related to store activity, including timestamps, images, and associated video evidence for operational and security monitoring.

**• Video Monitoring**
Access recorded video streams linked to alerts for comprehensive operational oversight and security review.

**• Daily Cash Register Reports**
Daily KPI reports displaying:
- Revenue metrics overview (total revenue, total transactions, average transaction value)
- Peak periods analysis with interactive line charts showing hourly revenue trends
- Staff and server performance table showing revenue contribution and transaction metrics

**• Financial Statistics & Analytics**
Financial analytics dashboard featuring:
- KPI cards for revenue, charges, and profit with evolution indicators and status colors
- Interactive doughnut chart for trend analysis
- Period-based analysis (daily, weekly, monthly views)
- Analysis insights and performance indicators
- Detailed charges breakdown view

**• Expense Management (Charges)**
Expense tracking and management with:
- Fixed charges tab for recurring expenses (personnel, utilities, etc.)
- Variable charges tab for one-time expenses with receipt tracking
- Category-based organization
- Historical expense viewing

**• Secure Access**
The application uses enterprise-grade authentication. Accounts are provided to partner stores after onboarding and business verification.

**• Bilingual Interface**
Storeyes supports both French and English, making it accessible to diverse retail operations.

Storeyes is intended for professional use by authorized partner businesses in the retail sector.

## App Review Information

### Notes for Review Team:

Storeyes is a B2B application designed exclusively for professional store monitoring and management. This application is not available for public registration.

**Account Access:**
- Accounts are created manually after business onboarding and verification
- Public registration is not available
- Access is restricted to verified partner businesses

**Demo Account Information:**
The provided demo account has full access to all features for review purposes:
- Alert Management: View and manage real-time alerts with timestamps, images, and video evidence
- Video Monitoring: Access recorded video streams linked to alerts
- Daily Cash Register Reports: View daily KPI reports showing revenue metrics (total revenue, transactions, average transaction value), peak periods analysis with hourly charts, and staff performance tables
- Statistics & Analytics: Access financial analytics dashboard with revenue, charges, and profit KPI cards, interactive doughnut charts, and analysis insights
- Expense Management: View and manage fixed charges and variable charges with receipt tracking

**Feature Overview:**
- Real-time alert monitoring with timestamps, images, and video evidence
- Daily cash register reports with revenue metrics, peak periods charts, and staff performance
- Financial statistics dashboard with KPI cards, trend charts, and analysis insights
- Expense management with fixed and variable charges tracking
- Bilingual interface (French/English)

**Technical Notes:**
- The application requires an active internet connection
- Video playback requires sufficient bandwidth
- Some features may display "no data" if the demo account has limited historical data

Please note that this is a professional B2B application and all features are designed for business use by authorized retail partners.
